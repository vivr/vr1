Several popular Python libraries support machine learning:

- **[scikit-learn](https://scikit-learn.org/):** a widely used ML toolkit with clear examples and documentation..  
- **[SciPy](https://scipy.org/):** useful for optimisation, statistics and scientific computing. Also includes support for sparse matrices.  
- **[NumPy](https://numpy.org/):** provides fast multi-dimensional arrays, commonly used as input to ML algorithms.

- **[Pandas](https://pandas.pydata.org/):** offers DataFrames for working with structured data. Great for cleaning, filtering and analysis.
- **[Matplotlib](https://matplotlib.org/):** a plotting library used to create charts such as histograms, bar charts and scatter plots.
- **[Seaborn](https://seaborn.pydata.org/):** built on top of matplotlib, offering attractive statistical graphics.
- **[Graphviz](https://graphviz.org/):** a tool for visualising graphs and structures like decision trees.
